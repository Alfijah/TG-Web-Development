Opdracht 1

  • Object

    Een object wordt ook wel een instance genoemd. Ieder object heeft een identiteit, een gedrag en een staat. De staat van een object wordt bewaard in field(variabelen), terwijl methods (functions) het gedrag weergeven. Een object wordt aangemaakt met het keyword "new".

  • Class

    Een class is de blauwdruk om objecten te maken. Om een class aan te maken, wordt de keyword "class" gebruikt.

  • Package

    Een package kan worden gezien als een map die gerelateerde classes groepeert. Packages worden gebruikt om naamconflicten te voorkomen en om beter onderhoudbare code te schrijven.

  • Constructor

    Een constructor is een method die wordt aangeroepenn wanneer een object wordt geinstantieerd (met de "new" keyword). Het doel van een constructor is om de nieuwe aangemaakte objecten te initialiseren voordat het wordt gebruikt.

  • Instance

    Een instance is een varibele die gedefinieerd worden in een class, buiten iedere methode, constructor, of code block. Ieder geinstantieerde object van een class heeft een aparte kopie of instance van die variabele. Een instance variabele behoort tot een class.

  • Method

    Een method is een codeblok die runt wanneer het wordt aangeroepen. Data ofwel parameters kunnen worden meegegven aan een method. Methods worden gebruikt om bepaalde activiteiten uit te voeren en staan ook bekend als functions.

Opdracht 2

  1. Waar wordt een local variabele gedeclareerd?

    Een local variabele wordt gedeclareerd in een method. Het is slechts toegankelijk binnen de desbetreffende method.

  2. Wat voor soort informatie sla je op in een instance variabele?

    Instance variabelen bevatten de staat van een object en waarden waarnaar verwezen wordt door methodes, constructor of codeblok.

  3. Wat voor soort informatie sla je op in een class variabele?

    Een class variabele bevat classmembers, ofwel de methods en variabelen die zijn gedefinieerd in de class.

  4. Wat is het voordeel van het gebruiken van een class variabele?

    Een class variabele bevat maar een soort waarde die door alle objecten in een class gebruikt kan worden.
