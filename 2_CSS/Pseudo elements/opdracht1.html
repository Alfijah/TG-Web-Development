Enkele voorbeelden van pseudo-elementen:

  1. ::after. Dit element wordt gebruikt om iets toe te voegen na de content van een element.
  2. ::before. Dit element wordt gebruikt om iets toe te voegen vóór de content van een element.
  3. ::first-letter. Dit element selecteert de eerste letter van elk element (voorbeeld in een p tag).
  4. ::first-line. Dit element selecteert de eerste regel van een element.
