Enkele voorbeelden van pseudo-elementen:

  1. ::marker. Dit element selecteert de bullets of nummers van een lijst/list tag.
  2. ::selection. Dit element selecteert een deel van een element/tekst dat geselecteerd wordt door de gebruiker door bijvoorbeeld te dubbelklikken.
  3. ::backdrop. Dit element creÃ«ert een backdrop die de gehele viewport beslaat en direct onder de dialog tag wordt weergegeven of een ander element dat de fullscreen modus opent met de Fullscreen API.
  4. ::cue. Dit element komt overeen met WebVTT-cues binnen een geselecteerd element. Dit kan gebruikt worden om captions en andere cues te stijlen in media met VTT tracks.
  5. ::file-selector-button. Dit element representeert de button van een input tag van type="file".
  6. ::part(). Dit element representeert ieder element binnenin een shadow tree met een overeenkomstig part attribute.
  7. ::placeholder. Dit element representeert de placeholder tekst in een input tag of textarea tag.
  8. ::slotted(). Dit element representeert ieder element dat werd geplaatst in een slot binnen een HTML template.
  9. ::spelling-error. Dit element representeert een tekst segment welke door de user agent gemarkeerd is als "incorrectly spelled".
  10. ::grammar-error. Dit element representeert een tekst segment welke door de user agent gemarkeerd is als "grammatically incorrect".
