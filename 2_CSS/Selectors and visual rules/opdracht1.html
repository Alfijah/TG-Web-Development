Er zijn 3 verschillende manieren om CSS in HTML in te laden.
Deze zijn als volgt:

  1. External CSS. Een externe stylesheet maken en deze koppelen aan het HTML document.

    Instructies:
    Maak een apart bestand aan met de extensie *.css. In het HTML bestand koppel je vervolgens het zojuist gemaakte CSS bestand door de code tussen de <head> </head> toe te voegen:
    <link rel="stylesheet" href="*.css">. Let er wel op dat je deze code toevoegt na de <title>Document</title>.

  2. Internal CSS stylesheet. Een stylesheet maken in het document zelf als interne stylesheet. De interne stylesheet wordt gedefinieerd in de <style> element, die binnen de <head> element.

    Instructies:
    <head>
        <style>
        body {
            color: blue;
        }
        </style>
    </head>

  3. Inline CSS. De stijl in een enkele HTML-tag veranderen.

    Instructies:
    <p style="color:red;">This is een zin.</p>
