Er zijn verschillende soorten selectors, bijvoorbeeld:

  • Type selectors
  • Descendant selectors
  • Class selectors

Type selectors

Deze selector selecteert alle elementen van het gegeven type in een document.

Voorbeeld:
p {font-size: 1em;}


Descendant selectors

Deze selector selecteert alle elementen die een descendant of ook wel child zijn van een specifiek element. Deze is heel handig om te gebruiken wanneer je bijvoorbeeld meerdere elementen hebt met dezelfde naam, maar specifiek één van ze wilt aanpassen.

Voorbeeld:
div p {font-size: 1.2em;}
Hieruit is te lezen dat alle <p> elementen binnen <div> elementen worden geselecteerd.


Class selectors

Deze selector selecteert elementen met een specifieke class attribute.

Voorbeeld:
.box { background-color: blue; }
