Cascade & Inheritance

Wanneer je met CSS werkt, is het belangrijk om de concepten Cascade en Inheritance te begrijpen. Deze worden nader toegelicht.

Cascading

Cascading in CSS verwijst naar een cascade van processen of styling regels. Deze regels zijn hiÃ«rarchisch van aard waarbij de stijling regels met een hogere prioriteit de regels met een lagere prioriteit overschrijven. Zie het voorbeeld om meer duidelijkheid te krijgen hoe deze regels worden geprioriteerd.
Voorbeeld:

p { color: red; }

p { color: black; }
<head>
     <link rel="stylesheet" href="style.css">
     <style>
          p {
              color: blue;
          }
     </style>
</head>
<body>
     <p style="color: green;">Cascading rules</p>
</body>

Cascading rules

Zoals afgebeeld in het voorbeeld is te zien dat het element p dezelfde property met elk een ander value is gegeven, namelijk red en black. Het is echter zo dat de tweede selector wordt gekozen namelijk datgene met de value black. Dat komt doordat volgens de styling regels de laatste selector de voorgaande selector(s) overschrijft, wanneer deze identiek zijn aan elkaar.

Verder is het belangrijk om te weten dat als er styling wordt toegepast in het html bestand, dat die de styling regels van het CSS bestand (met identieke selectors) zullen overschrijven. Zoals te zien in het voorbeeld zal de selector in het head element al de identieke selectors in de CSS overschrijven. De tekst krijgt dan de kleur blue.

Er is nog een manier om styling toe te voegen namelijk via inline CSS. De styling regel wordt direct toegepast binnen het element zoals te zien is in het voorbeeld, de selector met de value green. Wanneer deze regel wordt toegepast, gaat het alle overige regels ofwel overige identieke selectors overschrijven. De tekst krijgt uiteindelijk de kleur groen.

Inheritance

Inheritance wordt geassocieerd met hoe (child) elementen eigenschappen (values) erven van hun parent element.
Voorbeeld:

body { color: red; }

div { color: yellow; }

p { color: blue; }
<body>
     <div>
         <p>
         Cascading rules
         </p>
     </div>
</body>

Zoals te zien in het voorbeeld staan er drie soorten tags: body, div en p tag. De body tag is hier de parent, de div tag is de child en de p tag de grandchild. Als de body selector als enige styling regel aanwezig was, zou de value/kleur red gelden voor al de tags die binnen de body tag plaatsvinden. Als de div selector ook wel aanwezig was, zou de kleur yellow worden toegepast aan alle tags die binnen die div tag plaatsvinden. Als de p selector ook aanwezig was zou uiteindelijk de kleur/value blue gelden aan alle tags die eventueel nog toegevoegd zou kunnen worden binnen de p tag.
