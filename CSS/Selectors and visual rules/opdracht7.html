De CSS selectors die er bestaan, zijn:

  • Type selectors
    p {font-size: 0.9rem;}

  • Descendant selectors
    div p {color: blue;}

  • ID selector
    #box {border: yellow 1px solid;}

  • Class selectors
    .first {background-color: red;}

  • Adjacent selectors
    div + p {font-weight: bold;}

  • General selectors
    div ~ p {color: white;}

  • Universal selectors
    * {margin: 0;}

  • Attribute selector
    a[title] {color: purple;}
