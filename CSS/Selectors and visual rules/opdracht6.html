Er zijn verschillende selectors waaronder de volgende drie:

  • Child selector
  • Adjacent selector
  • Genereal selector

Child selector

Voorbeeld:
div>p {color: green;}
Het voordeel van deze selector is dat je heel specifiek kan kiezen welk element je wilt aanpassen, wanneer je meerdere elementen binnen een element hebt.

Adjacent selector

Voorbeeld:
div + p {color: green;}
Het voordeel van deze selector

General selector

Voorbeeld:
div ~ p {color: green;}
Het voordeel van deze selector
