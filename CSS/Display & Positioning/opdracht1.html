Er bestaan de volgende soorten position values:

  • Static: Het element wordt gepositioneerd volgens de normale flow van het document.
  • Absolute:Het element wordt relatief gepositioneerd ten opzichte van zijn eerst bovenliggende element.
  • Fixed: Het element wordt relatief gepositioneerd ten opzichte van het browser venster.
  • Relative: Het element wordt relatief gepositioneerd ten opzichte van zijn eigen positie.
  • Sticky: Het element scrollt mee met zijn container. Waneer het tijdens het scrollen de top van zijn container heeft bereikt, stopt het element met scrollen waardoor het altijd zichtbaar blijft.
  • Initial: Het element wordt ingesteld op zijn standaard waarde.
  • Inherit: Het element gebruikt de waarde van zijn bovenliggende element.

De default waarde van de position property is static.
